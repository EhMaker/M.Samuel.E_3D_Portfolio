- This is the **actively used** component. It renders as a modal overlay inside the 3D scene on the Home page.
- It's imported and displayed in [`Home.jsx`](src/pages/Home.jsx) when the user clicks the monitor mesh:

```jsx
{
  showProjects && (
    <ProjectsSection projects={projects} onClose={handleCloseSection} />
  );
}
```

### [`Projects`](src/pages/Projects.jsx)

- This is a **standalone page** component designed for a `/projects` route.
- However, looking at [`App.jsx`](src/App.jsx), **there is no route defined for `/projects`**:

```jsx
<Routes>
  <Route path="/" element={<Home />} />
  <Route path="/about" element={<About />} />
  <Route path="/contact" element={<Contact />} />
</Routes>
```
